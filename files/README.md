# Private Data Folder

Place private MoU source files in this folder when running the project locally.

The real college MoU records are intentionally ignored by Git because they may contain sensitive institutional data. The repository only includes `sample_public_mou.json`, a synthetic example that demonstrates the expected JSON shape.

Supported local files:

- `*.json`
- `*.md`
- `*.meta.json`

Do not commit real MoU records, generated exports, or private metadata.
